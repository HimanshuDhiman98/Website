Windows 11: Recent Update


Microsoft said on Tuesday that Windows 11 — the most recent version of its operating system — will be enhanced with a series of improvements aimed at hybrid work and enterprise clients. Microsoft will add tabbed and pinned file capabilities to Windows 11's File Explorer. Improved accessibility capabilities for people with impairments have also been highlighted, as well as the ability to effortlessly switch between a local Windows 11 PC and a Windows 365 Cloud PC. In addition, the business is introducing support for a service that automates Windows, Microsoft Edge, and Office upgrades.
